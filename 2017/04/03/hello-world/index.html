搭建步骤：安装 Node.js 和 hexo，

最后一步下载 next 主题 到 themes 目录，更改_config.yml 文件 theme: next .

更新详细步骤

安装 Node.js

  • 官网下载 source code，Binaries 试过不行。
  • Ubuntu下，需要root权限安装。 sudo ./configure; sudo make -j4; sudo make install
  • 测试 node -v，显示版本则安装成功。
  • 重启机子，必要。

安装 HEXO

  • sudo npm install -g hexo-cli
  • 初始化 mkdir BLOG; cd BLOG; hexo init . BLOG 为根目录。
  • 生成静态页面 hexo generate; 启动本地服务 bash hexo server
  • 浏览器输入http://localhost:4000, 可以查看博客了。但只能在本地用，下面推送到 Github.

Github

  • 建立Repository, 仓库名必须和用户名一样，格式：your_user_name.github.io。这块卡了好久。
  • 编辑_config.yml文件，来建立关联。注意冒号后面的空格，否则会报缩进不对的错误。

deploy:
type: git
repository: https://github.com/liuzhicheng00/liuzhicheng00.github.io.git
branch: master

  • 推送到远程仓库，然后在浏览器中输入https://liuzhicheng00.github.io/浏览。

npm install hexo-deployer-git –save
hexo generate
hexo deploy

适合做笔记，目前用的moeditor 界面好简洁。也可在线编辑，直接看效果，https://www.zybuluo.com/mdeditor#673406。
